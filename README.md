# Loofi Fedora Tweaks

A GUI application for Fedora 43 (KDE Plasma) tailored for the HP Elitebook 840 G8.

## Features

- **System Updates**: DNF, Flatpak, and Firmware updates.
- **Cleanup & Maintenance**: DNF cache cleaning, autoremove, journal vacuuming, SSD trim, RPM DB repair.
- **HP Elitebook Tweaks**: Power profile management, Audio service optimization, Battery charge limit control (80%/100%).
- **Essential Apps**: One-click installers for Chrome, VS Code, Steam, Discord, VLC.

## Installation

Run the application using the provided `run.sh` script or the desktop entry.

## Requirements

- Fedora 43 (KDE Plasma recommended)
- Python 3
- PyQt6
- polkit (pkexec)
